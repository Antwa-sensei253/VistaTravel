# VistaTravel

## Project Info

This project is a React application built with Vite, TypeScript, Tailwind CSS, and Shadcn UI.

For detailed information about features, architecture, and setup, please refer to the [Full Documentation](./DOCUMENTATION.md).

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Install dependencies
npm install
```

### Development

```sh
# Start the development server
npm run dev
```

### Build

```sh
# Build for production
npm run build
```

## Technologies

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
